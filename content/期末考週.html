影片連結:BG3-XML UI slider 控制功能

心得:

依照我們之前所學remote API ，將程式丟入小白窗，找到TOOLS接著按下GO，就會跑出一個頁面，可以控制joint1、joint2、joint3的拖曳鍵。這算是一種比較簡單的控制三軸的方法，但如果比較複雜的角度可能就做不出來了。

6.

影片連結:BG3-跨網路 Python remote API心得:

查看協同組員之IPV4，然後將程式的self.clientID' ..... '中的IPV4改成組員的IP，組員只需要在 CoppeliaSim 中的 Main script 裡增加一行程式 simRemoteApi.start(19999) ，在依照第四題拖進小白窗執行一樣動作便可，透過網路操控伺服器上的 uArm 機械手臂更有不同的感覺。
